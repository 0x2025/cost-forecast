# Cost Forecast Marketing Website

This is the professional marketing website for Cost Forecast, designed for B2B consultant clients.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Tailwind CSS v4 (Consulting Grade Theme)
- **Charts**: Recharts
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Getting Started

1. Navigate to the directory:
   ```bash
   cd websites/cost-forecast-marketing
   ```

2. Install dependencies (if not already installed):
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Key Features
- **Professional Theme**: Matches the "Consulting Grade" aesthetic of the main application.
- **Case Study**: "2025 Supply Chain Resilience" with interactive sensitivity and scenario charts.
- **SEO Optimized**: Meta tags and semantic structure.
